### Calendar Module in Python :
1. Python has a built-in module for Calendar Management, providing all the methods and operators to manage the Calendar functionalitites.
2. The Calendar module of Python follows `Gregorian Calendar standards`, and it contains all the classes, methods that are essential in managing the real-time calendar functionalities.

#### Month Method :
1. This method operated to print the specific month of the year.
    - calendar.month(Year, Month, Width, Lines)
    - calendar.month(2021, 9, 3, 1)
