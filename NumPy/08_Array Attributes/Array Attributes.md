## Understanding Array Attributes

1. NumPy provides several array attributes, that typically provide information about the array's structure and memory layout.
2. The list of Array attributes are:
    - ndarray.shape
    - ndarray.size
    - ndarray.dtype
    - ndarray.ndim
    - ndarray.itemsize
    - ndarray.nbytes
    - ndarray.T
    - ndarray.flags
    - ndarray.strides
    - ndarray.base
    - ndarray.real
    - ndarray.imag
    - ndarray.flat
    - ndarray.ctypes
    - ndarray.view()
    - ndarray.copy()
    - ndarray.tolist()
    - ndarray.data()
