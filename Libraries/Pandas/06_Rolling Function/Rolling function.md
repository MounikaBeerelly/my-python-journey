### Pandas `rolling()` function :

1. Pandas `rolling()` function provides the facility for rolling upon the data for window calculations.
2. The concept of Rolling is mostly used in data related to `Signal Processing` and `Time Series Analysis`.
3. Rolling helps in taking a window size of `N` at a time and then perform the required mathematical calculation.
4. The window size `N` identifies the `N` number of values to be considered at a time.
