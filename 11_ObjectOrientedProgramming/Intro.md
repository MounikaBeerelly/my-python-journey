### What is an Empty class in Python ?
- Empty classes are extendable
- **Basic Syntax :**
    ```
        class EmptyClass :
            pass
        
        emptyClassObject = EmptyClass() # Creating Instance of the class : object
    ```
- Print the structure of the class : `emptyClass.__dict__`
