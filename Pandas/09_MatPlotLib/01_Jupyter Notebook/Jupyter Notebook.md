## What is Jupyter Notebook ?

- Jupyter Notebook is an interactive coding environment where you can write and run Python code cell by cell.
- Great for:
    - Data analysis (Pandas, NumPy)
    - Machine learning
    - Visualizations (Matplotlib, Seaborn)
    - Documentation + code together

### How to Install Jupyter Notebook

1. **Using pip**
    - Open Command Prompt (CMD) or Anaconda Prompt.
    - Run:
        - `pip install notebook`
    - After installation, run:
        - `jupyter notebook` - This opens Jupyter in your browser.
    
2. **Install via Anaconda (If working with data science)**
    - Download Anaconda:
        - `https://www.anaconda.com/products/distribution`
    - Install it.
    - Open Anaconda Navigator
    - Click Launch under Jupyter Notebook.
    - Start Jupyter
        - `jupyter notebook` - It opens a tab in your browser showing your folders.
